

 	Detta program är byggt för Windowsdatorer.
	Dennis von Bargen garanterar icke att detta program fungerar
	på varken Mac eller Linux, eller något annat operativsystem.


Installation:

	Lägg mappen som innehåller denna textfil, SimpEc v1.0, samt undermappen 
	tables var du vill i din dator.
	Det är rekommenderat att ni lägger den i Program Files.


	Tables innehåller en exempeltabell för att demonstrera hur dessa är 
	tänkta att användas.
